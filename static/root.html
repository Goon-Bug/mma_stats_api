Welcome to the MMA API!

Get all data

To get all data add '/all' to the URL Example: 'https//:server_url/all'

Search by Nationality

To search by nationality, in the URL add '/nationality/{nationality}' Example: 'httpl://server_url/nationality/england'

Search by Name

To search by name, in the URL add '/name/{name}' Example: 'httpl://server_url/name/jon jones'

Search by Division

To search by division, in the URL add '/division/{division_name}' Example: 'https//:server_url/division/heavyweight'

Add any of these keywords after '/division/' to get the corresponding division stats (not case sensitive):

- heavyweight
- lightheavyweight
- middleweight
- welterweight
- lightweight
- featherweight
- bantamweight
- flyweight
- wfeatherweight
- wbantamweight
- wflyweight
- wstrawweight
- watomweight"
